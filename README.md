# Code for the analysis of mouse lung SRT data

Figurexx.R: code for generating results of figure xx.

RCTD_analysis.R: code for the RCTD deconvolution analysis.

SCENIC_analysis: code for the SCENIC gene regulatory network analysis.

## System requirements

The analysis scripts were run in R and Python environments on a Windows 11 workstation.
